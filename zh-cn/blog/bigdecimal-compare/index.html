BigDecimal 比较大小

介绍 Java 中 BigDecimal 类的大小和相等判断方法，使用 compareTo 进行精确比较。

2024/2/22 Java #Java

这个类是 Java 里精确计算的类，下面说一下 BigDecimal 大小、相等的判断。

BigDecimal a = new BigDecimal(1.0);
BigDecimal b = BigDecimal.valueOf(1.000);

一、比较方法

int result = a.compareTo(b);

二、比较结果

  • -1 小于
  • 0 等于
  • 1 大于

判断时以大于 0、小于 0、等于 0 为条件。
