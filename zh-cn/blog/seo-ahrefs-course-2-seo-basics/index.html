SEO学习笔记-Ahrefs公开课（二）：SEO的基础知识

SEO的好处、初期设置、关键词研究、页面SEO、链接建设、技术性SEO以及免费工具推荐。

2025/1/7 SEO学习笔记 #SEO#Ahrefs#SEO基础#学习笔记

SEO基础

SEO的好处是什么？

  • SEO可以带来更多流量
  • SEO可以带来稳定流量
  • SEO流量是”免费”的

如何加入谷歌排名？

  • 第一步，将你的站点地图提交给谷歌。
  • 第二步，创建一个免费的 Google Search Console 账户并提交你的站点地图。

SEO初期设置

SEO 是一个持续不断的过程，如果做得好，随着时间的流逝，你获胜的机会就越大。

选择不错的域名

好域名的一些属性：是你的公司名称、没有连字符、简短而令人难忘 尽量选择顶级域名，如 .com、.org等。

选择网站平台

托管平台可以一站式完成所有工作。他们托管你的网站，提供可以轻松安装和编辑的现成设计，并允许你创建和管理内容，而无需触摸代码。

选择优质的主机

  • 安全。确保主机商为你提供了免费的 SSL / TLS 证书、或支持 LetsEncrypt（提供免费TLS证书的非营利组织）。
  • 服务器位置。数据在服务器和访问者之间传输需要花费时间，因此最好选择距离目标客户近的主机位置。
  • 技术支持。 24/7 技术支持是最理想的。在注册之前，请先询问上述问题，以测试他们的技术支持可靠性。

创造积极的用户体验

  • 使用 HTTPS
  • 选择一个吸引人的设计
  • 保持移动端友好
  • 使用可读的字体大小
  • 避免弹出式窗口和广告
  • 确保加载速度快

创建站点逻辑结构

使用有逻辑的URL结构

如果你是从头开始建立新网站，则最好采用最清晰，最具描述性的结构。那就是文章名称。 在 URL 中使用日期也是一种不好的做法，毕竟你将来可能希望更新这个文章。 如果要优化现有网站，则最好不要更改 URL 结构——即使它不是理想的。这样做可能会破坏其他更重要的事情，从而对你的SEO产生更大的负面影响。

安装一个不错的SEO插件

我们使用 Yoast，但是 Rank Math 也很好。

SEO怎么做

关键词研究

  • 寻找关键词
  • 查看流量潜力
  • 检查排名难度

页面 SEO

页面 SEO 涉及创建用户正在寻找的内容类型、并对其进行 “优化” 以帮助搜索引擎更好地了解其含义。

  • 符合搜索意图
  • 选择一个描述性的URL别名
  • 编写引人注目的标题
  • 优化图片：给他们起个好名字、使用描述性替代文字、压缩他们

链接建设

  • 做客博客
  • 摩天楼技术：查找包含大量外链的相关内容、建立更出色的内容、请求别人链接到你的内容
  • 跟踪竞争对手的链接：了解竞争对手如何吸引外链的是链接建设策略的最佳方法。

技术性 SEO

我们建议你注册一个免费的 Ahrefs Webmaster Tools（Ahrefs站长工具）账户，并安排定期抓取你的网站。这些它会帮你检测 100 多个 SEO 问题，其中就包括许多常见的技术问题。

如何跟踪SEO效果

监控声音份额指标（SoV）

自然的声音份额是指你的品牌在搜索结果中的可见度。

跟踪自然流量带来的变化

免费的SEO工具

关键词研究工具

  • 谷歌关键词规划师：查找独特的关键词建议。
  • Keyword Generator（关键词生成器）：生成数百个关键词建议。
  • SERP Checker（SERP 检查）：根据前三个搜索结果检查该主题的搜索流量潜力。
  • Keyword Difficulty Checker（关键词难度检查）：检查关键词的关键词难度（Keyword Difficulty）得分。

页面SEO工具

  • Yoast SEO：将标题标签和元描述添加到 WordPress 的文章和页面。
  • Ahrefs Webmaster Tools（Ahrefs站长工具）：在网站上查找缺少的标题、元描述、替代文本的页面。

链接建设工具

  • Ahrefs’ Backlink Checker（外链检查）：查看指向任何网站或网页的前100个外链。
  • HARO：获取每天的链接机会（从技术上讲，它更多的是服务而不是工具）
  • Gmail：发送外联电子邮件以联系潜在客户。其实并不需要花哨的外推软件。

技术类SEO工具

  • Google Search Console：查找索引错误、页面速度问题等。
  • Ahrefs Webmaster Tools（Ahrefs站长工具）：帮助你找出100多个SEO问题，包括许多技术类问题。
