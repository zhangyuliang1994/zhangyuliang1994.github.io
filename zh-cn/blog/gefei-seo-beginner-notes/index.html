哥飞SEO学习笔记：新手入门

2026/3/6

最近在系统学习哥飞的新手入门课程，我把关键观点、方法论和能立刻上手的 SOP 做一次集中整理。核心就一句话：

种一棵树最好的时间是十年前，其次是现在。
做网站是长期积累，别等“准备好了”，先开始，边做边迭代。

一、海外工具站的发展前景：先短期练手，再长期布局

对大多数新手来说，最容易卡在“想做大项目，但能力/资源/耐心跟不上”。哥飞给的路径很清晰：

1）先做短期、for fun 的小需求练手

目的不是一上来就赚钱，而是快速跑通这条链路：

  • 怎么挖需求（关键词、场景、痛点）
  • 怎么做 SEO（站内结构、TDK、内容）
  • 怎么获取流量（搜索引擎为主）
  • 怎么推广（社媒/社区/外链）
  • 怎么变现（Adsense/联盟/订阅/询盘）

只要这条链路你跑通一次，后面会越来越快。

2）再做长期的、工作生活需要的工具站

长期项目要做“能持续更新、能持续扩展”的站：

  • 参考竞争对手
  • 在某些维度超过对手：更好用 / 更漂亮 / 更快 / 更全 / 更专注
  • 逐步把站做成“专业度更强”的内容与工具组合（后面会讲枢纽-辐射式结构）

二、怎么挖掘需求：从“看网站”到“拆关键词”再到“验证难度”

哥飞这套挖需求的方法，本质是用工具把“直觉”变成“数据”。

1）Similarweb：先看站值不值得研究

用它快速判断一个网站：

  • 流量大不大、趋势怎么样
  • 流量主要来自哪些国家
  • 流量渠道结构（搜索/社媒/直接/外链等）
  • 还能顺带发现相似站点（继续拓展选题）

2）Semrush：拆关键词、看分布、找机会

Semrush 的价值在于“拆解”：

  • 自然搜索流量、付费流量、反链
  • 关键词分布（哪些词带来流量、各自体量）
  • 关键词难度 KD、国家分布等

关键点：
当你发现一个站的大量流量来自某个“明确工具词”，而且这个词的竞争不强——你就能反推：

我注册一个更贴词的域名 + 举全站之力做这个词，很可能更容易上首页。

3）Google Trends：看趋势，避免做“下降通道”

用途很简单但很关键：

  • 判断这个需求是上升/稳定/下降
  • 多个关键词的相对热度对比（选更值得投入的）

4）Ahrefs：最终验证关键词难度（KD）

在 https://ahrefs.com/keyword-difficulty 做最后确认：

  • KD 很低（比如 9）且给出参考：只需要少量外链就可能进首页
  • 这类词更适合新站拿“快速正反馈”

课程里强调的思路是：先用 Similarweb/Semrush 找机会，再用 Ahrefs 验证值不值得做。

三、实战挖需求：一个判断“是否能抢首页”的小技巧

哥飞提到一个非常实用的经验：

同一个关键词竞争：域名 > 子域名 > 子目录 > 内页

所以当你在 Google 搜某个词时，如果前排结果很多是“内页”，往往意味着：

  • 这个词竞争可能没那么强
  • 你用“专门域名 + 首页/核心页”去打，反而更容易进第一页

再结合 Ahrefs 看 KD 和外链需求，就能快速决定要不要做。

另外还有一个细节很有启发：

  • 域名带 AI 在传播上可能更“省力”（更贴热点、更容易解释）
  • 举例：ai-coloring-pages.com 这种域名，关键词和 AI 叠加，天然更贴搜索意图

四、怎么制作工具站：AI + 模板，但“数据准备”是关键

这里是我认为最值得照抄的 SOP。

1）先做数据准备：不要让 AI 瞎编关键词

哥飞强调：不能直接让 GPT/Claude“拍脑袋写 Title/Description”。

正确顺序是：

  1. 用 Similarweb / Semrush 的关键词生成器
  2. 输入核心关键词，拿到真实搜索词列表
  3. 下载“语句匹配 + 相关关键词”两份表
  4. 合并、去重、按搜索量排序
  5. 取前 100 个关键词（只保留关键词本身，一行一个）

2）再让 AI 生成 SEO 友好的 Title/Description

把“网站定位 + 站名 + 核心词 + 100 个关键词”喂给 AI，让它：

  • 首页 Title 覆盖站名 + 核心词，符合长度规范
  • Description 用自然语言表达价值，不堆砌

这一步做完，你的首页 TDK 质量就会显著高于大多数新手站。

3）站内 SEO + 速度 + 变现布局

哥飞的实践方向很明确：

  • 重写/优化前后端，提升速度
  • 站内 SEO 做扎实（后面有详细清单）
  • Adsense 广告位布局：建议最多 3 个广告位（符合 Adsense 文档建议）

五、SEO 改造的大原则：旧 URL 不要动

这是非常重要的一条：

已经存在的 URL 不要改动。

原因很现实：

  • 已经被收录了
  • 外部可能已有外链指向
  • 改 URL 等于把积累清零（甚至引发大量 404）

在这个前提下，SEO 改造主要做两类事：

1）挖需求，新增页面

新增页面可以是：

  • 小工具页
  • 内容页（教程、解释、清单、对比、FAQ）

目标是围绕潜在用户需求，把相关问题尽可能覆盖掉。

2）找问题，优化老页面

按哥飞提到的检查项，我整理成一份“站内 SEO 自检清单”：

  • TDK 是否规范（Title/Description 长度与表达）
  • 内链是否合理（从首页到内页尽量不超过 4 次点击）
  • 是否有 sitemap.xml
  • 是否设置 canonical
  • H1/H2/H3 是否合理（建议一个页面一个 H1）
  • 图片 alt 是否完善
  • 站外链接是否需要 nofollow
  • robots meta 是否正确（该收录的别误 noindex）

六、工具清单：Ahrefs 免费工具矩阵（够新手用很久）

课程里把 Ahrefs 的免费工具拆得很细，我按用途归类一下，方便你以后查：

关键词方向

  • Free Keyword Generator
  • Keyword Difficulty Checker
  • SERP Checker
  • Keyword Rank Checker

流量与权威

  • Website Traffic Checker
  • Website Authority Checker

外链与死链

  • Backlink Checker
  • Broken Link Checker

平台关键词

  • Amazon Keyword Tool
  • Bing Keyword Tool
  • YouTube Keyword Tool

辅助工具

  • Ahrefs SEO Toolbar（强烈推荐，做网页诊断很直观）
  • AI Writing Tools
  • WordPress SEO Plugin（做 WP 可用）

七、内容策略的几个关键认知：泛流量没钱途，要做精准流量

这里我自己感触很深：

  • 泛娱乐流量往往很难变现
  • 精准流量（尤其是 B 端）才更有价值
    传统企业依然需要搜索流量，你用内容去承接需求，就能形成稳定的转化入口

同时，AI 写文章是可行的，但前提是：

  • 不是乱写
  • 要基于企业服务范围/产品信息
  • 围绕潜在客户会搜的词
  • 真正解决痛点，提升“信任度”

八、关于排名因素：新鲜度、信任度、外链、参与度、专业度

课程里还提到一组对“做工具站的人”很实用的启发（我用自己的话总结）：

  • 新鲜度越来越重要 → 工具站也要更新、要有内容（所以要加博客）
  • 信任度成为关键 → 胡编/AI 胡编都会被打低信任，内容要认真做
  • 外链依然有效 → 但不要买那种“80 美元 100 条垃圾外链”
  • 用户参与度重要 → 满足搜索意图，用户停留时间更长，排名更有利
  • 专业度重要（枢纽-辐射式结构） → 首页/栏目页做“枢纽”，细分关键词页做“辐射”

九、我接下来的行动清单（照着做就能推进）

最后给自己定一个可执行的节奏：

  1. 每天用 Similarweb / Semrush / Ahrefs 看 3 个站，保持数据敏感度
  2. 每周选 1 个低 KD 且趋势稳定/上升的关键词，做一个最小可用工具页
  3. 每个工具站都加博客：每周至少更新 2-3 篇内容（教程/FAQ/对比/清单）
  4. 做站内 SEO 自检（TDK、H 标签、alt、canonical、sitemap、内链深度）
  5. 尽量不动旧 URL，只新增、只优化
  6. 追求“快速正反馈”：优先做一年内新出现的新词（更容易上首页）
